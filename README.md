# edges
 
**This is a new magazine.** Its name is Edges. It wants work from *you*.

Edges is a magazine dwelling in the space between, exploring what exists in connection, finding out what is buzzing between all these cables. Edges asks what becomes of the social network without the app, what becomes of correspondence when paper is added back in. What if the social network exists between *you* and *me*, as we share, discuss, observe, and feel? Edges wants work (essays, stories, poems, visual, etc.) around this theme.

Edges magazine is a global community paper for all sorts of sharing, connecting, writing and learning.
